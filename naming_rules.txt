# Naming conventions for the AutoFE framework
# Library: AutoFE
# Version: 1.0.0
# ------------------------------------------

## 1. Class naming rules
# --------------------------------------------

### 1.1 Base Classes
Pattern: [Base][FeatureType]
Example:
  - BaseGenerator

### 1.2 Feature Generators
Pattern: [Action][FeatureType]Generator
Example:
  - AggregationFeatureGenerator
  - StatisticalFeatureGenerator
